# 🏥 Clinic Patient Management System

A simple **Windows Forms application** developed in **C#** using **.NET Framework** in Microsoft Visual Studio.  
The system allows clinic staff to manage patient records efficiently.

---

## 📌 Features

- ➕ Add new patients
- 🔍 Search patients 
- 📋 View patient list
- ❌ Delete patient records
- ✔ Input validation for age and phone number

---

## 🖥 Technologies Used

- C#
- .NET Framework
- Windows Forms
- Microsoft Visual Studio
- Git & GitHub
